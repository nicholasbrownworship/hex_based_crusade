# Bastior Prime Hex Map (v1)

This is a **starter** map folder intended for early playtests.

Files:
- `map_layout_v1.json` — the hex grid and which hexes contain installations
- `spawns_v1.json` — faction spawn hexes
- `tokens_v1.json` — starting army tokens (optional)

Coordinate system:
- Uses **axial hex coordinates** (q, r).
